Java Script

Package and task managers​

  • NPM

  • Gulp

  • Grunt

Two crucial things from my point of view during building a site or an app are automation and organization. Automation is when we force a computer to execute some task for us. The organization is clearly written code (many separate files which are related to a certain part of a website ) and just one bundled file (css & js) which browser will load quickly to satisfied high performances. To accomplish all of that here I decide to use Webpack, which is really great tool. With which I made a different setup for the development and production environment.
